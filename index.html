HTML Cheatsheet




HTML Semantics

Tag Name Description To Clipboard
<header> Header Element Represents a container for introductory content or a set of navigational links.  
<nav> Nav Element Defines a set of navigation links.
<section> Section Element Defines a section in a document.
<article> Article Element Specifies independent, self-contained content.
<aside> Aside Element Defines some content aside from the content it is placed in (like a sidebar).
<footer> Footer Element Defines a footer for a document or section.



HTML Tables

Tag Name Description To Clipboard
<table> Table Element Defines an HTML table element.
<thead> THead Element Defines a group for header elements. Used in conjunction with <tbody>, <tfooter>.
<tbody> TBody Element Defines a group for body elements.Used in conjunction with <thead>, <tfooter>.
<th> Th Element Defines a table heading element.
<tr> Tr Element Defines a table row element.
<td> Td Element Defines a table data element.
